# Car Price Prediction using Machine Learning

This project predicts car prices using supervised machine learning.

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Algorithms Used
- Linear Regression
- Random Forest Regressor

## Workflow
1. Data Cleaning
2. Encoding
3. Model Training
4. Make Prediction
5. Evaluation

## Dataset
Car dataset containing features like:
- Car Name
- Year
- Selling Price
- Fuel Type
- Transmission
